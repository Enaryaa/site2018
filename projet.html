Mes projets

Pour décembre nous avons dû, en binôme créer un Memory dans le cadre de l'école. Pour cela nous avons eu un délai de 1 mois.
Afin de réaliser le projet plusieurs contraintes ont été posée; un timer devait apparaitre, plusieurs modes de difficultés devaient être créé et un mode triche a été mis en place. L'utilisation de Makefile était également obligatoire.
Nous avons créé notre propre graphique, avec les différents fonds, les cartes ainsi que les boutons avec photoshop.

Notre programme se présente en un menu principal avec les différents mode de difficulté et un bouton menant aux règles du jeu

Chaque fenêtre de jeu se présente de la même manière; on trouve le timer en haut à gauche, les cartes en fonction de la difficulté (par exemple la grille normale est de 5x6 ), ensuite à droite on indique les différents boutons, q pour quitter et t pour le mode triche.

une fois toutes les cartes trouvées, le jeu s'arrête et ouvre un nouveau fond qui est l'écran de victoire, avec un bouton qui mène au menu principal.

Nous avons dû utiliser la bibliothèque graphique de l'IUT. Ainsi par exemple, pour créer la grille de jeu, il a fallu stocker les sprites des cartes dans un tableau en 1 exemplaire afin de récupérer l'id donné par la fonction ChargerSprite et de reconnaitre par la suite les cartes.

On créé ensuite un second tableau afin de mettre en 2 exemplaires, à des places aléatoires chaque chaque et les placer sur la grille

Il faut que l'utilisateur clique deux fois, sur deux cartes différentes afin de comparer leur id, si les id sont pareils, les cartes restent retournées et le score d'incrémente, sinon ils sont recachés.
Le score permet de déterminer si je le jeu est fini ou non, le score dépend alors du nombre de paire présente dans la grille. Le score s'incrémente à chaque paire trouvée jusqu'à atteindre le nombre maximum. Le jeu sera alors fini et l'écran de victoire apparaît.
